# Desafío-Allura-Store
Durante este desafío, se pretendió ayudar al Sr. Juan a decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento. Se analizaron datos de ventas, rendimiento y reseñas de las 4 tiendas de Alura Store. Se identificó la tienda menos eficiente y se presentó una recomendación final basada en los datos.

El código se divide en 7 secciones:

### 1. Análisis de facturación
En esta sección se calculó el ingreso total de cada tienda. Sumando los valores de la columna Precio de cada conjunto de datos de la tienda para estimar los ingresos.

### 2. Ventas por categoría
En esta sección se calculó la cantidad de productos vendidos por categoría en cada tienda. Se gruparon los datos por categoría y se contabilizó el número de ventas de cada tipo, mostrando las categorías más populares de cada tienda.

### 3. Calificación promedio de la tienda
En esta sección se calculó las calificaciones promedio de los clientes para cada tienda. El objetivo fue conocer la satisfacción del cliente con los productos vendidos.

### 4. Productos más y menos vendidos
En esta sección se  identificó qué productos fueron los más vendidos y los menos vendidos en cada tienda. Además, se visualizó los resultados para que quede claro qué productos destacaron en ventas en cada tienda.

### 5. Envío promedio por tienda
En esta sección se calculó el costo de envío promedio para cada tienda. El objetivo fue comprender cuánto se gasta, en promedio, en el envío de cada tienda.  Sabiendo que el cliente paga el envío.

### 6. Gráficos
En esta sección se generó un gráfico Stackplot de la biblioteca Matplotlib. El Stackplot refleja, de forma normalizada, el **ingreso total de cada tienda**, el **envío promedio de cada tienda**, así como su **calificación promedio** sintetizando las secciones que consideré más relevantes para visualizar. 

### 7. Informe final
En esta sección se redactó el informe final que identifica a la tienda con menor desempeño, de acuerdo al análisis de datos, para ponerla en venta.
