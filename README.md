# Desafío-Allura-Store
Durante este desafío, se pretendió ayudar al Sr. Juan a decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento. Se analizaron datos de ventas, rendimiento y reseñas de las 4 tiendas de Alura Store. Se identificó la tienda menos eficiente y se presentó una recomendación final basada en los datos.
